# Tori

A local-first inspiration & media library for macOS. Save images, videos, links, and audio in one beautiful place.

![Tori App](https://github.com/emrnc/tori/raw/main/preview.png)

## Features

- **Local-first** — all files stored on your machine, nothing in the cloud
- **Beautiful grid** — masonry layout that adapts to your content
- **Multiple types** — images, videos, links with OG preview, audio
- **Spaces** — custom collections to organize your library
- **Favorites & Archive** — keep what matters, archive the rest
- **Command Menu** — Cmd+K to search and navigate fast
- **Dark & Light mode** — follows your system or set manually
- **List view** — switch between grid and list
- **iCloud sync** — coming soon

## Built with

- [Electron](https://electronjs.org)
- [React](https://react.dev) + TypeScript
- [Vite](https://vitejs.dev)

## Getting started

```bash
# Install dependencies
npm install

# Run in development
npm run dev

# Build for production
npm run build
```

## Philosophy

Tori is free. All features are free. Always.

The app is built for designers, artists, writers — anyone who collects visual inspiration and wants a beautiful, fast, private place to keep it.
